
레퍼런스 변수(Reference variable)는 프로그래밍에서 변수가 메모리 상에 위치한 객체나 데이터 구조를 참조하는 데 사용되는 변수입니다. 다른 말로는 참조 변수(Reference variable) 또는 참조자(Reference)라고도 부릅니다.

레퍼런스 변수는 객체 지향 프로그래밍에서 흔히 사용되며, 객체의 메모리 주소를 저장하고 해당 객체에 접근하는 데 사용됩니다. 이 변수는 객체를 직접 포함하거나 저장하지 않고, 단지 객체가 위치한 메모리 주소를 가리킵니다.

레퍼런스 변수의 주요 특징은 다음과 같습니다:

객체 참조: 레퍼런스 변수는 객체를 참조하기 위해 사용됩니다. 객체를 생성하고 해당 객체의 메모리 주소를 레퍼런스 변수에 할당하여 객체에 접근할 수 있습니다.

값의 할당과 전달: 레퍼런스 변수에는 객체의 메모리 주소가 저장되므로, 변수 간에 값의 할당이 이루어지면 실제로는 같은 객체를 참조하게 됩니다. 따라서 한 변수를 수정하면 다른 변수에도 동일한 변경 내용이 반영됩니다.

Null 값 할당: 레퍼런스 변수에는 null 값을 할당할 수 있습니다. 이는 해당 변수가 어떤 객체도 참조하지 않음을 나타냅니다.

레퍼런스 변수는 다양한 프로그래밍 언어에서 사용됩니다. 예를 들어, Java와 C++에서는 클래스 인스턴스를 가리키기 위해 레퍼런스 변수를 사용하며, Python에서는 객체를 참조하기 위해 변수를 활용합니다. 각 언어에 따라 레퍼런스 변수를 다루는 방식과 규칙이 조금씩 다를 수 있습니다.
